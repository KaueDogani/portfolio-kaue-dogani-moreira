### 📂 Portfólio de Projetos

 
# ♻️ Recyclo

## 📝 Descrição
O Recyclo é um projeto que busca incentivar hábitos sustentáveis através de um sistema com elementos de gamificação, recompensas e acompanhamento de progresso.

O objetivo é tornar o processo de reciclagem mais atrativo e acessível para usuários comuns.

---

## 🚀 Tecnologias Utilizadas
* HTML
* CSS
* JavaScript

---

## 📊 Objetivo do Projeto
- Incentivar reciclagem
- Criar engajamento com sistema de recompensas
- Desenvolver habilidades em front-end
